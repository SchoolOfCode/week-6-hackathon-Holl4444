
# Space Escape!

The Challenge:

Creating a CodeWars like challenge with a theme that is continous across team members!
Using Unit testing.


## Tech Stack

- Node
- Vitest
- Javascript


## Running Tests

To run tests, run the following command

```bash
  npm run test
```

To check  coverage, run the following command

```bash
  npm run coverage
```
